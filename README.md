# Bittorrent-cli

Bittorrent-cli is a command-line interface Bittorrent client.

![Demo](./vhs/demo.gif)

## Run

Instal golang:

https://go.dev/doc/install

Run application:

```bash
go run . --torrent [Path to torrent file] --download [Path to download folder]
```
